💡 프로젝트 개요

사용자의 여행 스타일을 설문조사하여 AI가 맞춤형 여행지를 추천하고, 세부 일정을 계획하여 동행자와 공유할 수 있는 웹 서비스입니다.

  • 1. 데이터베이스 설계 및 최적화 (AWS Aurora)
    - 사용자(User), 설문(Survey), 일정(Plan) 등 핵심 엔티티의 1:N, N:M 관계 정의 및 ERD 설계.
    - AWS Aurora(MySQL Compatible) 인스턴스를 생성하고 보안 그룹(Inbound/Outbound) 설정.
    - 정규화를 통해 데이터 중복을 최소화하고 무결성 유지.
  • 2. FastAPI 백엔드 개발 및 트러블 슈팅
    - SQLAlchemy ORM을 활용하여 Python 객체와 DB 테이블 매핑.
    - Google Gemini API 연동 중 발생한 Timeout 이슈를 비동기 처리로 해결.
    - 로컬(Windows)과 배포 서버(Linux) 간의 환경 차이로 인한 모듈 에러 해결.
  • 3. AWS 클라우드 인프라 & 배포
    - AWS EC2(Ubuntu) 서버 구축 및 Docker 컨테이너를 활용한 웹 서버 구동.
    - RDS와의 안정적인 통신을 위한 VPC 및 서브넷 네트워크 설정.
    - GitHub를 통한 버전 관리 및 협업 프로세스 주도.
